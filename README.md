## YAHFAXposed


YAHFAXposed is an Android hook framework written like Xposed,based on YAHFA.

Currently it supports:
- Android 7.0(API 24)
- Android 7.1(API 25)
- Android 8.0(API 26)
- Android 8.1(API 27)
- Android 9(API 28)
- Android 10(API 29)

with ABI:
- armeabi-v7a
- arm64-v8a
